# Ruby Tricks

### Random number generator

Generate random float between 0.0 and 1.0

    $ rand
    => 0.3162109656780836

Generate random int between 0 and 3

    $ rand(3)
    => 1

Generate random int between 100 and 102

    $ rand(100..102)
    => 102

Generate random int between 100 and 101

    $ rand(100...102)
    => 101
